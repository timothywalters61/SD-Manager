    Product Terms



    Projects

    A collaborative collection of work to achieve a goal comprising of a project owner and project developer

    Sprints

    A short time period (usually 2-3 weeks) where a project development team completes a set amount of work

    User Story

    A natural description of features of the project being developed

    Task

    A defined piece of work that is assigned to one or more people


    Home Page



    The Navigation

    The Navigation Bar contains a projects, invitations and logout tabs. Hover over the invitations tab to see how many projects you're invited to collaborate on. To log out of the SD manager, simply click on the logout tab

    Project Display

    All the projects you have joined will be displayed in blocks on your home page, click on them to view details such as sprints, user stories and tasks. If you have not joined any projects, a message saying "No Projects Yet" will be displayed

    Create Project

    >To create a new project. click on the create project button and add developers of your choice by email. You will be required to name and give a short description of your project.


    Project Owner/Developer Pages



    Sprints

    The Navigation Bar contains a projects, invitations and logout tabs. Hover over the invitations tab to see how many projects you're invited to collaborate on. To log out of the SD manager, simply click on the logout tab

    Project Display

    All the sprints, sprint due and end dates, team members and gitlink pertaining to a certain project will be displayed. A button to create a new sprint will be displayed as well for project owners to further progress of the project group.


    Sprint Section



    Sprint Display

    The Project Name, Sprint, team and gitlink displayed. Members of a team can create a new user story by clicking on the "create a new user story" button. User Stories will then display with their name, description , acceptance criteria, points and a "view tasks" at the bottom of the block. Click on "View tasks" button to see tasks for a specific user story.


    User Stories Section



    User Story and Task Display

    The Project Name, Sprint, team and gitlink displayed along with a blue block containing the name of the user story accessed. Members of a team can create a new task by clicking on the "Add a task" button. The added tasks will then be displayed at the bottom at the bottom of the block.
